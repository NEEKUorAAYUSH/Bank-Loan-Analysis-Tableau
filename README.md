# 📊 Bank Loan Analysis Dashboard (Tableau + SQL)

## 🔍 Project Overview
This project analyzes a bank's loan portfolio to evaluate lending performance, risk distribution, and borrower behavior. The goal is to derive actionable insights for improving loan approval strategies and risk management.

---

## 🛠 Tools & Technologies
- Tableau (Dashboard Visualization)
- PostgreSQL / SQL (Data Analysis)
- Excel (Data Cleaning)

---

## 📊 Key Metrics (KPIs)
- Total Loan Applications: 38.6K
- Total Funded Amount: $435.8M
- Total Amount Received: $473.1M
- Average Interest Rate: 12.0%
- Average DTI: 13.3%

---

## 📈 Key Insights

### 1. Loan Quality
- 86.2% loans are **Good Loans**
- Only 13.8% are **Bad Loans**
👉 Indicates a strong lending portfolio with low default risk

---

### 2. Revenue Performance
- Total Amount Received > Total Funded Amount  
👉 Bank is generating **positive returns**

---

### 3. Loan Purpose Analysis
- Majority loans are for **Debt Consolidation**
👉 Customers primarily borrow to manage existing debt

---

### 4. Borrower Stability
- Higher employment length → higher loan approval
👉 Stable jobs reduce risk

---

### 5. Geographic Trends
- Certain states dominate loan distribution
👉 Opportunity for regional strategy optimization
